# 💡 Ý tưởng - Content Crawler

## Mục tiêu chính
- Crawl nội dung bài trắc nghiệm từ https://app.formation-tcfcanada.com/
- Lấy dữ liệu 4 kỹ năng: Nghe (Compréhension Orale), Nói (Expression Orale), Đọc (Compréhension Écrite), Viết (Expression Écrite)
- Xuất dữ liệu dạng cấu trúc để dùng cho website riêng

## Ý tưởng thêm
- [ ] Lưu câu hỏi + đáp án + giải thích
- [ ] Lưu file audio (nếu có) cho phần nghe
- [ ] Phân loại theo level (A1, A2, B1, B2, C1, C2)
- [ ] Xuất ra JSON/CSV để dễ import vào website
